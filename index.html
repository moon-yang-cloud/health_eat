定位附近真实店铺 · 按距离与营养智能筛选 · 远离纯碳水

⚙️ 首次使用：填入高德地图 Key（约5分钟）

  1. 打开 高德开放平台 lbs.amap.com，注册并登录。
  2. 进入「应用管理 → 我的应用 → 创建新应用」，再「添加 Key」，服务平台选 Web端(JS API)。
  3. 把拿到的 Key 和 安全密钥(securityJsCode) 填进 index.html 顶部的配置区，保存即可。

提示：JS API 必须同时配置 Key 和安全密钥两项，缺一不可。
